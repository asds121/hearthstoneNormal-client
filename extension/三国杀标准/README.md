# 三国杀标准扩展

## 扩展简介

三国杀标准扩展是基于无名杀游戏框架开发的三国杀标准模式扩展包，包含了标准模式下的所有卡牌、角色和游戏模式。

### 主要内容

- **卡牌系统**：包含标准模式下的所有卡牌定义
- **角色系统**：包含标准模式下的所有武将角色
- **游戏模式**：支持身份场、单挑模式和斗地主模式
- **技能系统**：整合了卡牌技能和角色技能

## 扩展结构

```
extension/三国杀标准/
├── assets.json          # 扩展资源配置
├── info.json            # 扩展元信息
├── LICENSE              # 扩展许可证
├── package.js           # 扩展包配置
├── README.md            # 扩展说明文档
├── assembly/            # 应用程序组件
│   └── application.js   # 应用程序类
├── audio/               # 音频资源
├── card/                # 卡牌定义
│   └── standard/        # 标准卡牌
├── character/           # 角色定义
│   └── standard/        # 标准武将
├── image/               # 图片资源
└── mode/                # 游戏模式定义
    ├── identity/        # 身份场模式
    ├── single.js        # 单挑模式
    └── doudizhu.js      # 斗地主模式
```

## 安装说明

1. 将整个 `三国杀标准` 目录复制到项目的 `extension/` 目录下
2. 扩展会被自动加载，无需额外配置

## 使用说明

1. 启动游戏客户端
2. 系统会自动加载已配置的扩展
3. 在游戏模式选择界面选择相应的游戏模式
4. 开始游戏
